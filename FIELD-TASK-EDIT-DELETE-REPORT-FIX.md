# Field Task Edit/Delete and Report Fix

Updated the field task workflow with:

- Task creator can edit and delete field tasks.
- Employee/update author can edit and delete their own work updates.
- Task creator can also edit/delete updates for the task.
- Employee report moved behind a Show/Hide button.
- Report filters added: last 7 days, 15 days, 30 days, this month, this year, all time.
- Export now downloads a real CSV file with `.csv` extension so Excel no longer shows file format mismatch/corruption warning.
- Report columns: Time, Zone, engineer name, TID, POS serial, Merchant address, service type, Remarks.
- Remarks column uses the completed work update first input field.
